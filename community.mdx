## Contributing to Nexu

Nexu is open-source (MIT) and actively welcoming contributors. Whether you want to fix a bug, improve documentation, add a new channel integration, or build a custom skill, contributions of all sizes are valued.

<Note>
Before diving in, read the [Contributing guide](/dev/contributing) for the full workflow — branching conventions, commit style, PR format, and what to expect during review.
</Note>

### Good First Issue program

If you are new to the codebase, the [Good First Issue list](https://github.com/nexu-io/nexu/labels/good-first-issue) is the best place to start. These issues are deliberately scoped to a single area and come with AI Prompt templates to help you ramp up quickly. The team does their best to provide guidance and timely feedback for first-time contributors and `good-first-issue` claimers.

Nexu runs an open-source co-creation program — contribute code, earn points, and get featured on the leaderboard. See the contributor rewards and support details linked from the [Contributing guide](/dev/contributing).

<Tip>
Not ready to write code yet? Documentation improvements, issue triage, and sharing Nexu in your community are all genuinely useful contributions.
</Tip>
